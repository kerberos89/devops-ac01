# Requesitos
#De acordo com a análise de requisitos é fundamental que o e-commerce possua um tela de cadastro e login de acesso para permitir a interação do usuário e controlar o fluxo de compra/entrega (distribuição), além disso o catálogo de produtos tem por objetivo garantir a visualização e especificar as características dos produtos ofertados.
#Como fluxo de compra/entrega, será realizada a integração com meio de pagamento online e integração com o sistema de logistica.

# Design
# Modelo conceitual da aplicação

# O software deve conter um layout único, em uma página, o usuário deve fornecer os dados inicialmente para apresentação do resultado no final. 
# Deve conter integração com as redes sociais, com opções de comentários, compartilhamentos no final da página, junto a uma integração com google maps indicando o endereço da loja.  
# Ao acessar a página internet deve ser ativo um programa de chatterbot com usuário. Deve conter funcionalidades como: consulta de CEP, consulta de frete e disponibilidade do produto.
# O site deve conter uma sessão para que os usuários possam entrar em contato, ao lado com depoimentos de antigos compradores com suas avaliações.
# Deve conter também uma galeria com imagens em três fileiras sobre o produto e consumidores na loja. 


#Codificação

#Desenvolvimento Teórico

Para o desenvolvimento do software em sua parte backend, levando em conta a utilização de novas tecnologias 
e a necessidade do desenvolvimento de uma plataforma de e-commerce, será utilizada a linguagem
Python através do framework Django, que conta com grande fluidez, boa curva de aprendizagem e
rapidez no desenvolvimento;

Para o frontend, serão utilizados recursos nativos do CSS 3 que, recentemente, foram  adaptados aos navegadores
mais atualizados do mercado. Dentre elas temos CSS Grid Layout e Flex Box, ambas técnicas contam com grande possibilidade de 
otimização de UX e taxa de responsividade para diversos dispositivos;

Para o banco de dados será utilizado o MySQL, administrador comumente usado em aplicações web e de fácil administração
com ferramentas como o SQL Workbench.
